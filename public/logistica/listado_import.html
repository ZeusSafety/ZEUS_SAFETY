Procedimiento de Importaciones Logística

Proceso de Gestión de Importaciones

1. Registro de Importación

• Se registra la importación con todos los datos básicos

• Se asigna un número de despacho único

• Se especifica el tipo de carga y productos

2. Seguimiento de Estados

• ETA: Estimated Time of Arrival - Tiempo estimado de llegada

• TRANSITO: En camino hacia destino

• PRODUCCION: En proceso de producción/manufactura

• PENDIENTE: Esperando procesamiento

3. Canales de Prioridad

• VERDE: Prioridad normal

• NARANJA: Prioridad media

• ROJO: Prioridad alta

4. Gestión de Incidencias

• Registrar cualquier problema durante el proceso

• Actualizar fechas de recepción cuando corresponda

• Documentar incidencias para seguimiento

5. Actualización de Datos

• Utilizar el botón "Actualizar" para modificar información

• Actualizar fecha de recepción cuando los productos lleguen

• Marcar incidencias si las hay

Notas Importantes:

• Mantener actualizada la información para un seguimiento eficiente

• Los documentos PDF deben estar siempre disponibles

• Comunicar cualquier cambio de estado inmediatamente

• El sistema permite filtrar por fechas y número de despacho
